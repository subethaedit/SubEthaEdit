Tastatur-Kurzbefehle
 

Drücken Sie ⌥⎋ (alt-escape) oder F5 zum Aktivieren der automatischen Vervollständigung.

Halten Sie die Wahltaste gedrückt während Sie einen Text auswählen, oder drücken Sie ⇧⌘B nachdem Sie Text ausgewählt haben, um Blockedit zu aktivieren.

Drücken Sie ⎋ (escape) zum Beenden von Blockedit.

Drücken Sie ⌘R zum Anzeigen der Web-Vorschau.

Drücken Sie ⌘E zum Benutzen der Auswahl als Suchtext und drücken Sie anschließend ⌥⌘F zum Ausführen der "Alle Suchen"-Funktion.

Drücken Sie ⇧⌘I zum Anzeigen der unsichtbaren Zeichen im Text.

Drücken Sie ⇧⌘U zum Öffnen der URL, in der sich die Einfügemarke gerade befindet.

Drücken Sie ⌃⌥⌘P, um die Auswahl durch ein Shell-Kommando zu pipen.

Drücken Sie ⇧⌘T zum Öffnen einen Terminal-Fensters mit dem Pfad der Datei.
    HTML Modus
 

Drücken Sie ⌥⌘. zum Schließen des letzten offenen Tags.

Drücken Sie ⇧⌘L zum Einfügen eines Links. Wenn die Zwischenablage eine URL enthält wird diese benutzt. Der Link-Text entspricht der aktuellen Auswahl.

    Objective-C Modus
 

Drücken Sie ⇧⌘L zum Einfügen eines NSLog-Aufrufs, der den aktuellen Funktions- oder Methodennamen enthält.

Drücken Sie ⇧⌘O zum Öffnen der entsprechenden .h oder .m Datei.

Drücken Sie ⇧⌥⌘O zum Öffnen der Datei, die in der aktuellen Zeile referenziert wird.

Drücken Sie ⇧⌘X zum Öffnen des zugehörigen Xcode Projekts.

Drücken Sie ⌘B zum Bauen mit Xcode.

Drücken Sie ⌘/ zum Ein- oder Auskommentieren.

Drücken Sie ⇧⌘H zum Nachschlagen der Auswahl in AppKiDo.
    Perl Modus
 

Drücken Sie ⇧⌘B zum Überprüfen der Syntax.

Drücken Sie ⌃⌘R zum Ausführen des Skripts im Terminal.

Drücken Sie ⇧⌘H zum Nachschlagen der Auswahl in perldoc.perl.org.
    PHP Modus
 

Drücken Sie ⇧⌘B zum Überprüfen der Syntax.

Drücken Sie ⇧⌘L zum Einfügen eines Links. Wenn die Zwischenablage eine URL enthält wird diese benutzt. Der Link-Text entspricht der aktuellen Auswahl.

Drücken Sie ⇧⌘H zum Nachschlagen der Auswahl in php.net.
    Python Modus
 

Drücken Sie ⇧⌘B zum Überprüfen der Syntax.

Drücken Sie ⌃⌘R zum Ausführen des Skripts im Terminal.

Drücken Sie ⇧⌘H zum Nachschlagen der Auswahl in python.net.
    Ruby Modus
 

Drücken Sie ⇧⌘B zum Überprüfen der Syntax.

Drücken Sie ⌃⌘R zum Ausführen des Skripts im Terminal.
    XML Modus
 

Drücken Sie ⌥⌘. zum Schließen des letzten offenen Tags.

Drücken Sie ⇧⌘B zum Überprüfen der Wohlgeformtheit.
