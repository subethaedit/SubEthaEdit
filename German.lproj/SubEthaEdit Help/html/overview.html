Überblick
 

In diesem Abschnitt erfahren Sie mehr über SubEthaEdit.

Dokumente ankündigen
 Damit andere Benutzer Ihre Dokumente sehen können müssen Sie diese ankündingen. Dazu klicken Sie einfach das Ankündigen-Symbol in der Symbolleiste des Dokuments oder drücken ⇧⌘E. Nun sehen die anderen Benutzer Ihr Dokument in ihrem Rendezvous-Fenster und können mit einem Doppelklick darauf teilnehmen.

Benutzer einladen
Andere Benutzer können Sie in Ihre Dokumente einladen indem Sie sie einfach aus dem Rendezvous oder Internet-Fenster in Ihr Dokument ziehen.

An Dokumenten teilnehmen
Wenn Sie an einem Dokument in Ihrem Subnetz teilnehmen wollen wählen Sie dieses durch Doppelklicken in Ihrem Rendezvous-Fenster (⌘K) aus. Um eine Verbindung über das Internet herzustellen, öffnen Sie das Internet-Fenster (⇧⌘K) und geben den Rechnernamen oder die IP-Adresse des Benutzers an, der das gewünschte Dokument freigegeben hat.

Sichtbar oder Unsichtbar?
Wenn Sie es anderen ermöglichen wollen Sie einzuladen wählen Sie "Immer sichtbar" im Menü unterhalb Ihres Names im Rendezvous-Fenster. Für den Fall, dass Sie nicht gesehen werden wollen wählen Sie "Sichtbar, wenn Dokumente angekündigt sind".
